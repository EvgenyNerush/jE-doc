Detailed Description

template<size_t i, typename... Ts>
struct zipWith< i, Ts >

A C++ functor which provides zipWith::operator() to zip two tuples element-wise up to a given index with a given function. The first argument is used to return the result.

Examples:

zipWith<2, f>((1, 2, 3), (1, 1, 1)) for auto f = ()[int x, int y]{return x + y;} changes the first tuple to (2, 3, 4).

zipWith<1, g>((1, 2, 3), (4, 5, 6)) for auto g = ()[int x, int y]{return x * y;} changes the first tuple to (4, 10, 3).
